# type
- 자바스크립트 : 다이나믹 타입 (런타임 상황에서만 앎)
- 타입스크립트 : 스태틱 타입 (개발할 때 앎)

# Primitive Type
- 오브젝트/레퍼런스 vs 프리미티브형
   - boolean, number, string, symbol, null, undefined
     - 리터럴 값으로 서브타입으로 표현
     - 래퍼 객체로 만들 수 있음 
        - Boolean(true), String('world')
           - 래퍼객체로 사용하는 것이랑 타입으로 사용하는 것을 구별
